Monoid

Keith A. Lewis

March 28, 2023

Abstract
Concatination

A moniod is a set M with a binary operation m\colon M\times M\to M that is associative and has an identity element e.

A binary operation m is associative if m(m(a,b),c) = m(a,m(b,c)), a,b,c\in M. Writing ab for m(a,b) this means (ab)c = a(bc), so writing abc in unambiguous.

An identity element e satisfies ea = a = ae for a\in M.

If an associative binary operation lacks an identity we can define one.

Exercise. If ea = a = ea, a\in M, then (ea)b = ab = e(ab), (ae)b = ab = a(be) and (ab)e = ab = a(be).

This shows adding an identiy preserves associativity.

Exercise. If e'\in M satisfies e'a = a = ae', a\in M, then e' = e.

This shows the monoid identity is uniqure.

Examples

Multiplication is a monoid with identity element 1. Addition is a monoid with identity element 0. Maximum is a monoid with identity element -\infty. Minimum is a monoid with identity element \infty. String concatenation is a monoid with identity element the empty string. The arrows of a category with a single object form a monoid.

Rational

Monoids are the basis of the map-reduce algorithm. If a_j are elements of a monoid then the calculation of a_1\cdots a_n can be partitioned into seperate calculations A_1 = a_1\cdots a_{k_0}, A_2 = a_{k_0 + 1}\cdots a_{k_1}, \dots, a_{k_j + 1}\cdots a_n.
