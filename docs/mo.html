Monoid

Keith A. Lewis

A moniod is a set M with a binary operation ,\colon M\times M\to M that is associative and has an identity element \epsilon_M, so (s,t),u = s,(t,u) makes writing s,t,u unambiguous using infix notation s,t = ,(s,t).

The Kleene star of M is M^* = \{m_0,\ldots,m_{k-1}\mid m_j\in M, 0\le < k\}, the set of finite ordered sequences of elements of M including \epsilon_M when k = 0.

If f\colon M\to N is a function between two monoids we can define the fold of f byf^*\colon M^*\to N by f(m_0,\ldots,m_{k-1}) = f(m_0),\ldots, f(m_{k-1}) and f^*(\epsilon_M) = \epsilon_N.
