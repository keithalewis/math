One Period Model

Keith A. Lewis

March 28, 2023

Abstract
Buy now, sell later.

Let I be the set of instruments available for trading over a period of time and Î© be the set of what can happen over the period. The prices at the beginning of the period are x\in\mathit{R}^I. The prices at the end of the period are X\colon\Omega\to \mathit{R}^I where X(\omega)\in\mathit{R}^I are the instrument prices if \omega\in\Omega occurred.

A position \xi\colon\mathit{R}^I\to\mathit{R} is a linear function from prices to the value of a position. It corresponds to the number of shares in each instrument being purchased. The cost of acquiring the position is \xi x. The value at the end of the period is \xi X(\omega) if \omega\in\Omega occurred. The space of positions is denoted \mathit{R}_I = (\mathit{R}^I)^*.

Trading account activity is modelled by A\colon R_I\to\mathit{R}\oplus B(\Omega) where A\xi = -\xi x\oplus\xi X. The initial trade \xi causes \xi x to be debited from the account at the beginning of the period and \xi X(\omega) will be credited when the trade is unwound at the end of the period.

Exercise. Show the adjoint A^*\colon \mathit{R}\oplus ba(\Omega)\to\mathit{R}^I is A^*(\pi\oplus\Pi) = -x\pi + \int_\Omega X\,d\Pi.

Arbitrage exists in a one-period model if there is a position \xi\in\mathit{R}_I with \xi x < 0 and \xi X(\omega)\ge0, for all \omega\in\Omega.

Define a cone in \mathit{R}\oplus B(\Omega) by \mathcal{P} = \{y\oplus Y\mid y > 0, Y\ge0\} where y\in\mathit{R} and Y\in B(\Omega).

Exercise. There is no arbitrage if and only if \operatorname{ran}A\cap\mathcal{P} = \emptyset.

There exists a hyperplane in \mathit{R}\oplus B(\Omega) containing \operatorname{ran}A that does not intersect \mathcal{P}. Every hyperplane has the form \mathcal{H} = \operatorname{ker}\pi\oplus\Pi for some \pi\in\mathit{R}, \Pi\in B(\omega).

Exercise. Show \mathcal{H} = \operatorname{ker}1\oplus \Delta where \Delta = \Pi/\pi.

Hint: Show \pi\not=0.

Exercise. Show x = \int_\Omega X\,d\Delta.

Hint: \operatorname{ran}A\subseteq \mathcal{H}

If \zeta X = 1 for some \zeta\in\mathit{R}_I then a zero coupon bond is available. Its price is D = \zeta x.

Exercise. Show x = E[X]D for some probability measure.

Remarks

The dual of a vector space V is the set of all linear functionals V^*.

For \xi\in V^*, \operatorname{ker}\xi is a hyperplane in V.

If V has a topology we consider only continuous linear functionals.

V\oplus W, (V\oplus W)^*\cong V^*\oplus W^*.
