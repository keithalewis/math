Getting from A to Z

Keith A. Lewis

To prove ‘if A then Z’ or, equivalently, ‘A\Rightarrow B’ write down A. Below that write a true statement and the substitutions to apply to get the next statement B. If the statements P and P\Rightarrow Q occur in the derivation then the statement Q can be written (modus ponens). The proof is done when the last statement is Z.

Theorem. If A is an abelian group then x + x = x implies x = 0, for any x\in A.

Proof:

x + x = x

\qquad\{a = b \Rightarrow a + c = b + c\mid a\leftarrow x + x, b\leftarrow x, c\leftarrow(-x)\}

x + x = x\Rightarrow x + x + (-x) = x + (-x)

\qquad\{\text{modus ponens}\}

x + \underline{x + (-x)} = \underline{x + (-x)}

\qquad\{\underline{a + (-a)} = \overline{0}\mid a\leftarrow x\} (inverse)

\underline{x + \overline{0}} = \overline{0}

\qquad\{\underline{a + 0} = \overline{a}\mid a\leftarrow x\} (identity)

\overline{x} = 0

QED (quod erat demonstrandum)

To prove ‘A = \cdots = Z’ write down A. Below that write a true statement and the substitutions to apply to get the next statement =B. The equality is established when the last statement is =Z.

E[\underline{(X - E[X])^2}]

\qquad\{\underline{(a - b)^2} = \overline{a^2 - 2ab + b^2} \mid a\leftarrow X, b\leftarrow E[X]\}

= \underline{E[\overline{X^2 - 2XE[X] + E[X]^2}]}

\qquad\{\underline{E[A - B + C]} = \overline{E[A]} - \overline{\overline{E[B]}} + \overline{\overline{\overline{E[C]}}}\mid A\leftarrow X^2, B\leftarrow 2XE[X], C\leftarrow E[X]^2\}

= \overline{E[X^2]} - \overline{\overline{\underline{E[2XE[X]]}}} + \overline{\overline{\overline{\underline{\underline{E[E[X]^2]]}}}}}

\qquad\{\underline{E[cA]} = \overline{cE[A]}, \underline{\underline{E[b]}} = \overline{\overline{b}}\mid c\leftarrow 2E[X], A\leftarrow X, b\leftarrow E[X]\} (using XE[X] = E[X]X)

= E[X^2] - \overline{2\underline{E[X]E[X]}} + \overline{\overline{E[X]^2]}}

\qquad\{\underline{aa} = \overline{a^2}\} (exponent)

= E[X^2] - 2\overline{E[X]^2} + E[X]^2

\qquad\{ac + bc = (a + b)c\} (distributive)

= E[X^2] + (-2 + 1)E[X]^2

= E[X^2] - E[X]^2.

This establishes E[(X - E[X])^2] = E[X^2] - E[X]^2.
