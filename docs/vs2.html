Vector Space

Keith A. Lewis

March 3, 2024

Abstract
Abelian group with scalar multiplication satisfying the distributed law.

A vector is a function from an index set I to the real numbers, x\colon I\to\boldsymbol{R}. Functions can be multiplied by a scalar, (ax)(i) = (xa)(i) = a(x(i)), and added, (x + y)(i) = x(i) + y(i), a\in\boldsymbol{R}, i,j\in I.

If A and B are sets then the set exponential B^A = \{f\colon A\to B\} is the set of all functions from A to B.

Exercise. If A and B are finite show #(B^A) = #B^{#A}.

Hint: #A is the number of elements in set A.

Define the standard basis e_i\in\boldsymbol{R}^I by e_i(j) = 1 if i = j and e_i(j) = 0 if i\not=j.
