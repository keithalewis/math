Fréchet Derivative

Keith A. Lewis

January 22, 2024

Abstract
Infinite dimensional derivatives

The Fréchet derivative is the derivative of a function between normed vector spaces.

If f\colon\boldsymbol{R}\to\boldsymbol{R} then the derivative of f at x is f'(x) = \lim_{h\to 0}(f(x + h) - f(x))/h. The function f can be approximated near x by a line with slope f'(x).

The limit can be written using little o notation as f(x + h) = f(x) + f'(x)h + o(h). The derivative is actually a linear functional h\mapsto f'(x)h.

If F\colon X\to Y where X and Y are normed linear spaces then the Fréchet derivative of F at x is {DF\colon X\to\mathcal{B}(X,Y)} defined by F(x + h) = F(x) + DF(x)h + o(\|h\|) for x,h\in X where \mathcal{B}(X,Y) is the set of bounded linear operators from X to Y. The function F can be approximated near x by the linear operator DF(x).

The dual of a normed linear space X is X^* = \mathcal{B}(X,\boldsymbol{R}).
