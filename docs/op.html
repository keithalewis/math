Option Pricing

Keith A. Lewis

European option valuation involves calculating the expected value of the option payoff as a function of the underlying at expiration. Greeks are derivatives of the value with respect to model parameters. This short note derives formulas for these that can be used for any positive underlying.

Share Measure

Let Y be the positive, random price of some underlying instrument at option expiration. The (forward) value of an option paying \nu(Y) in some currency at expiration is E[\nu(Y)] = \int \nu(Y)\,dP. We can also consider the payoff in terms of shares of Y, \nu_s(Y) = \nu(Y)Y/E[Y]. If we receive \nu_s(Y) shares of Y at expiration we can convert those at price Y to \nu(Y) in the currency.

Share measure P_s is defined by dP_s/dP = Y/E[Y]. We write E_s[f(Y)] = E[f(Y)Y/E[Y] for expectation under the share measure.

Parameters

The forward is y = E[Y] and the vol squared is s^2 = \operatorname{Var}(\log Y). Every positive random variable can be written Y = e^{m + sX} for constants m, s and X having mean 0 and variance 1. Using y = E[Y] = e^m E[e^{sX}] we see e^m = y e^{-Œ∫(s)} where Œ∫(s) = \log E[e^{sX}] is the cumulant of X. Hence Y = ye^{sX - Œ∫(s)} is parameterized by y, s, and a mean 0 variance 1 random variable X.

Greeks are the derivatives of value with respect to forward and vol.¬†If you prefer a different parameterization y = y(z,t), s = s(z,t) the chain rule can be applied to get greeks in terms of z and t. Any positive random variable Y can be parameterized using the forward f and vol s by Y = f e^{s X - Œ∫(s)}, where Œ∫(s) = \log E[e^{s X}] is the cumulant of X. Value and Greeks

Let X be any random variable with mean 0 and variance 1 and define Y = fŒµ_s(X) where Œµ_s(x) = e^{sx - Œ∫(s)}. The Esscher transform X_s of X is defined by P(X_s\le x) = E[1(X\le x)Œµ_s(X)] so E[g(X_s)] = E_s[g(X)] = E[g(X)Œµ_s(X)] for any measurable function g.

The (forward) value of an option paying ŒΩ(Y) at expiration is v = E[ŒΩ(Y)].

Delta is the derivative of value with respect to forward \frac{dv}{df} = E[ŒΩ'(Y)dY/df] = E[ŒΩ'(Y)Y/f] = E_s[ŒΩ'(Y)].

Gamma is the second derivative of value with respect to forward \frac{d^2v}{df^2} = \frac{d}{df} E[ŒΩ'(Y)Y/f] = E[ŒΩ''(Y)(Y/f)^2] = E_s[ŒΩ''(Y)Y]/f since d(Y/f)/df = 0.

Exercise. Show d^nv/df^n = E[ŒΩ^{(n)}(Y)(Y/f)^n] = E_s[ŒΩ^{(n)}(Y)Y^{n-1}]/f^{n-1}.

Vega is the derivative of value with respect to vol \frac{dv}{ds} = E[ŒΩ'(Y)dY/ds] = E[ŒΩ'(Y)Y(X - Œ∫'(s))] = fE_s[ŒΩ'(Y)(X - Œ∫'(s))].

The inverse of option value as a function of vol is the implied vol.

Spot

It is straightforward to convert these formulas to spot values and corresponding greeks. If the spot price of the underlying is u the forward is f = e^{rt}u where r is the risk-free continuously compounded interest rate and t is the time in years to expiration. The underlying at expiration is U = Y.

The spot value is v_0 = e^{-rt}E[ŒΩ(U)] and spot delta is dv_0/du = e^{-rt}E[ŒΩ'(U) dU/du] = e^{-rt}E[ŒΩ'(F) (dY/df) df/du] = e^{-rt}E[ŒΩ'(F) (dY/df) e^{rt}] = dv/df.

The spot and forward delta are equal but the spot gamma is d^2v_0/du^2 = d(dv/df)/du = (d^2v/df^2) df/du = e^{rt} d^2v/df^2.

The spot vega is dv_0/ds = e^{-rt}E[ŒΩ'(U) dU/ds] = e^{-rt}E[ŒΩ'(Y) dY/ds] = e^{-rt}dv/ds.

Put and Call

A put option pays ŒΩ(Y) = (k - Y)^+ = \max\{k - Y,0\} at expiration and has value p = E[(k - Y)^+]. A call option pays ŒΩ(Y) = (Y - k)^+ at expiration and has value c = E[(Y - k)^+]. Note (Y - k)^+ - (k - Y)^+ = Y - k is a forward with strike k so all models satisfy put-call parity: c - p = f - k. Call delta is dc/df = dp/df + 1 and call gamma equals put gamma d^2c/df^2 = d^2p/df^2. We also have dc/ds - dp/ds = 0 so call vega equals put vega.

Note Y \le y(x) iff X \le x(y) since dŒµ_s(x)/dx > 0. The value of a put is \begin{aligned} p &= E[(k - Y)^+] \\ &= E[(k - Y)1(Y\le k)] \\ &= k P(Y \le k) - E[Y 1(Y \le k)] \\ &= k P(Y \le k) - fP_s(Y \le k) \\ &= k Œ¶(x(k)) - f Œ¶_s(x(k)). \\ \end{aligned}

Put delta is \frac{dp}{df} = -Œ¶_s(x(k)).

Gamma for either a put or call is \frac{d^2p}{df^2} = E[Œ¥_k(Y)(Y/f)^2] = œà(k)(k/f)^2 = œÜ(x(k))k/f^2s = œÜ_s(x(k))/fs.

Vega for a put is \frac{dp}{ds} = -E[1(Y\le k)Y(X - Œ∫'(s))] = -f\frac{d}{ds}Œ¶_s(x(k)).

Digital

A digital put has payoff ŒΩ(Y) = 1(Y \le k) and a digital call has payoff ŒΩ(Y) = 1(Y > k) with values p = P(Y \le k) = Œ¶(x(k)), c = P(Y > k) = 1 - Œ¶(x(k)).

Digital put delta \begin{aligned} \frac{dp}{df} &= -E[\delta_k(Y)(Y/f)] \\ &= -\psi(k)k/f \\ &= -(\phi(x(k)/fs)k/f \\ &= -\phi(x(k))k/f^2s \\ &= -\phi_s(x(k))/fs \\ \end{aligned}

Digital gamma is \begin{aligned} \frac{d^2p}{df^2} &= -E_s[\delta_k'(Y)Y]/f \\ &= -\psi_s'(k)k/f \\ &= -\phi'(x(k))/f^2s^2 \end{aligned} where we use \psi_s(y) = \phi(x)/fs so \psi_s'(y) = \phi'(x)/fks^2.

Digital put vega is \begin{aligned} \frac{dp}{ds} &= -E_s[\delta_k(Y)Y(X - s)] \\ &= -\psi(k)k(x(k) - s) \\ &= -(\phi(x(k))/ks)k(x(k) - s) \\ &= -\phi(x(k))(x(k) - s)/s \\ \end{aligned}

Since p + c = 1 we have d^nc/df^n = -d^np/df^n and d^nc/ds^n = -d^np/ds^n for all n > 0.

Black Model

We use the above to derive the standard Black-Scholes/Merton formulas for value and greeks. In the Black model Y = fe^{œÉB_t - œÉ^2t/2} where f is the forward, œÉ is the volatility, and B_t is Brownian motion at time t. There is really no need to drag in Brownian motion to compute an option value, we only use the fact B_t is normally distributed with mean 0 and variance t. There is also no need to drag in t, let s = œÉ\sqrt{t} and X be standard normal so Y = fe^{sX - s^2/2} has the same distribution.

Recall if X is standard normal then E[e^{Œº + œÉ X}] = e^{Œº + œÉ^2/2} and E[g(X)e^{s X - s^2/2}] = E[g(X + s)] for any function g and s\in ùëπ. Note dŒ¶_s(x)/ds = -œÜ(x - s) = -œÜ_s(x).
