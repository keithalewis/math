Notation

Let’s give names to things.

Product

If X and Y are sets then X\times Y = \{(x,y)\mid x\in X, y\in Y\} is the cartesian produce of X and Y.

The projections \pi_0\colon X\times Y\to X where \pi_0(x,y) = a and \pi_1\colon X\times Y\to Y where \pi_1(x,y) = y characterize the cartesian product. If p_0\colon Z\to X and p_1\colon Z\to Y there exists p\colon Z\to X\times Y with \pi_0p = p_0 and \pi_1p = p_1.

Union

If X and Y are sets then X\sqcup Y = \{(i,z)\mid\text{where }z\in X, i = 0 \text{ or } z\in Y, i = 1\} is the disjoint union of X and Y.

The inclusions \nu_0\colon X\to X\sqcup Y where \nu_0(x) = (0,x) and \nu_1\colon Y\to X\sqcup Y where \nu_0(y) = (1,y) characterize the disjoint union. If n_0\colon X\to Z and n_1\colon Y\to Z there exists n\colon X\sqcup Y\to Z with n\nu_0 = n_0 and n\nu_1 = n_1.

Exponential

If X and Y are sets the exponential Y^X = \{X\to Y\} = \{f\colon X\to Y\} is the set of functions from X to Y.

Evaluation

The evaluation function e_Y^X = e\colon Y^X\times X\to Y is e(f,x) = f(x)\in Y, f\in Y^X, x\in X.

Curry

The curry function \gamma\colon \{X\times Y\to Z\}\to \{X\to\{Y\to Z\}\} is (\gamma(f)x)y = f(x,y)\in Z, f\in \{X\times Y\to Z\}, x\in X, y\in Y.

The uncurry function \gamma^*\colon \{X\to\{Y\to Z\}\}\to \{X\times Y\to Z\} is \gamma^*(f)(x,y) = f(x,y)\in Z, f\in \{X\to\{Y\to Z\}\}.

Natural numbers \bm{N}= \{0, 1, \ldots\}.

\iota n = \{0, \ldots, n - 1\}, n\in\bm{N}.

\iota\colon\bm{N}\to\bm{N}^* where \bm{N}^* = \sqcup_{n\ge0}\bm{N}^n. \iota n\in\bm{N}^n.

Remarks

Every expression is a list of data or functions in Polish notation.

Expressions are parsed right to left.

Data is pushed on the call stack.

Functions know what is on the call stack.
