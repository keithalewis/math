Time Change of Brownain Motion.

Consider right continuous piecewise constant functions c\colon[0,\infty)\to\mathbf{N}. Parameterize c by \{(t_j,c_j)\} where t_j are increasing. Note c(t) = t_j when t_j\le t < t_{j+1}.

Let W_n be standard random walk. Let V_t = c_n W_{t_n} be time changed random walk. \Var(V_t) = c_n^2 t_n. Let V_n = V_{t_n}.

\Delta X_n = \mu(n, W_n)\,\Delta t_n + \sigma(n, W_n)\,\Delta W_n.

// fill in tree
template<class X>
inline X E(const function<X(X)>& f, size_t n, const X* x, X* v)
{
    if (n == 0) {
        return std::numeric_limits<X>::quiet_NaN();
    }

    // last time in binomial tree
    size_t t = (n * (n - 1))/2;
    for (size_t i = 0; i < n; ++i) {
        v[t + i] = f(x[t + i]);
    }
    do {
        t -= n;
        --n;
        for (size_t i = 0; i < n; ++i) {
            X fti = (v[t + i] + v[t + i + 1])/2;
            v[t + i] = std::max(fti, f(x[t + i]));
        }
    } while (n);

    return v[0];
}
