Let \boldsymbol{{F}} be a field and the set exponential {\boldsymbol{{F}}^I = \{x\colon I\to\boldsymbol{{F}}\} = \{I\to\boldsymbol{{F}}\}} be the set of all functions from the set I to \boldsymbol{{F}}. Define scalar multiplication \boldsymbol{{F}}\times\boldsymbol{{F}}^I\to\boldsymbol{{F}}^I where {(a,x)\mapsto ax = xa} by {(ax)(i) = a(x(i)) = (xa)(i)} and vector addition {\boldsymbol{{F}}^I\times\boldsymbol{{F}}^I\to\boldsymbol{{F}}^I} where {(x,y)\mapsto x + y} by {(x + y)(i) = x(i) + y(i)}, i\in I. We say these are defined pointwise since x,y\in\boldsymbol{{F}}^I are equal if and only if x(i) = y(i) for all i\in I.

Exercise. Show x + y = y + x for x,y\in\boldsymbol{{F}}^I.

Hint Evaluate at i\in I and use the fact field addition is commutative.

Solution We have (x + y)(i) = x(i) + y(i) = y(i) + x(i) = (y + x)(i) for all i\in I.

Define \boldsymbol{{0}}\in\boldsymbol{{F}}^I by \boldsymbol{{0}}(i) = 0 for i\in I. We write \boldsymbol{{0}}^I to indicate the domain is \boldsymbol{{F}}^I. Note {\boldsymbol{{0}}^I\not=\boldsymbol{{0}}^J} if {I\not=J}.

Exercise. Show x + \boldsymbol{{0}}= x for all x\in\boldsymbol{{F}}^I.

Define -x\in\boldsymbol{{F}}^I by (-x)(i) = -x(i) for x\in\boldsymbol{{F}}^I and i\in I.

Exercise. Show -x = (-1)x and x + (-x) = \boldsymbol{{0}} for x\in\boldsymbol{{F}}^I.

This shows \boldsymbol{{F}}^I is an abelian group under vector addition with additive identity \boldsymbol{{0}}.

The standard basis of \boldsymbol{{F}}^I is the set of vectors e_i\colon I\to\boldsymbol{{F}} defined by e_i(j) = 1(i = j), i,j\in I, where 1(i = j) = 1 if i = j and 1(i = j) = 0 if i\not=j. This is also called the Kronecker delta \delta_{ij} = 1(i = j).

Exercise. Show x = \sum_{i\in I} x(i) e_i, x\in\boldsymbol{{F}}^I, if I is finite.

Hint. Compute (\sum_{i\in I} x(i) e_i)(j) = \sum_{i\in I} x(i) e_i(j).

Solution We have (\sum_{i\in I} x(i) e_i)(j) = \sum_{i\in I} x(i) e_i(j) = \sum_{i\in I} x(i)1(i = j) = x(j).

This shows every x\in\boldsymbol{{F}}^I is a linear combination of the standard basis vectors.

Exercise. Show if \sum_{i\in I} a_i e_i = \boldsymbol{{0}} then a_i = 0 for all i\in I.

Hint: Evaluate at j\in I.

Any set of vectors with this property is called independent

The dual space of \boldsymbol{{F}}^I is (\boldsymbol{{F}}^I)^* = \boldsymbol{{F}}^{e_I} where e_I = \{e_i\mid i\in I\}. For x\in\boldsymbol{{F}}^I define x^*\in\boldsymbol{{F}}^{e_I} by x^*(e_i) = x(i), i\in I.

Exercise. Show (ax)^* = ax^* and (x + y)^* = x^* + y^* for a\in\boldsymbol{{F}}, x,y\in\boldsymbol{{F}}^I.

Exercise. Show x^* = \boldsymbol{{0}} implies x = \boldsymbol{{0}} for x\in\boldsymbol{{F}}^I and (\boldsymbol{{F}}^I)^* = \{x^*\mid x\in\boldsymbol{{F}}^I\}.

Any function T\colon\boldsymbol{{F}}^I\to\boldsymbol{{F}}^J satisfying T(ax) = aTx and T(x + y) = Tx + Ty is called a linear operator.

The set exponential \boldsymbol{{F}}^I can be identifed with the cartesian product of I copies of \boldsymbol{{F}}.

Define projections e_i^*\colon\boldsymbol{{F}}^I\to\boldsymbol{{F}} by e_i^*(x) = x(i).

Exercise. Show if p_i\colon V\to\boldsymbol{{F}}, i\in I, then there exists p\colon V\to\boldsymbol{{F}}^I with p_i = e_i^*p.

Hint: Define (pv)(i) = p_i(v) and note (pv)(i) = e_i^*(pv).

Vector Space Axioms

Define \boldsymbol{{0}}\in\boldsymbol{{F}}^I by \boldsymbol{{0}}(i) = 0 for i\in I.

Exercise. Show 0x = \boldsymbol{{0}} for x\in\boldsymbol{{F}}^I.

Exercise. Show x + \boldsymbol{{0}}= x = \boldsymbol{{0}}+ x for all x\in\boldsymbol{{F}}^I.

This shows \boldsymbol{{0}} is the identity element for vector addition.

Exercise. Show x + y = y + x for x, y\in\boldsymbol{{F}}^I.

Vector addition is commutative.

Exercise. Show (a + b)x = ax + bx and a(x + y) = ax + ay for a,b\in\boldsymbol{{F}} and x,y\in\boldsymbol{{F}}^I.

Hint. Consider (a + b)x(i) and a(x + y)(i).

This establishes the distributive laws satisfied by scalar multiplication and vector addition.

Exercise. Show -x = -1x satisfies x + (-x) = \boldsymbol{{0}}.

Hint. Consider x + (-x) = 1x + (-1x) = (1 + (-1))x.

This shows -x is the additive inverse of x\in\boldsymbol{{F}}^I and vector addition is a commutative group.

Exercise. Show a(bx) = (ab)x for a,b\in\boldsymbol{{F}}, x\in\boldsymbol{{F}}^n.

Hint: Use the fact multiplication is associative in \boldsymbol{{F}}.

Abstract Vector Space

Any set V with a scalar multiplication and vector addition satisfying the above axioms is a vector space. We will eventually prove every vector space is isomorphic to \boldsymbol{{F}}^I for some set I but proofs involving only the axioms are more insightful.

A linear operator T\colon\boldsymbol{{F}}^I\to\boldsymbol{{F}}^J is a function satisfying T(ax + y) = aT(x) + T(y), a\in\boldsymbol{{F}}, x,y\in\boldsymbol{{F}}^I. The set of all such operators is denoted [\boldsymbol{{F}}^I\to\boldsymbol{{F}}^J]. It is a vector space with scalar multiplication (aT)(x) = a(T(x)) and vector addition (T + U)(x) = Tx + Ux, a\in\boldsymbol{{F}}, x\in\boldsymbol{{F}}^I, T,U\in[\boldsymbol{{F}}^I\to\boldsymbol{{F}}^J].

Since e_i\in\boldsymbol{{F}}^I we have Te_i = \sum_{j\in I} (Te_i)(j)e_j and we define t(i,j) = Te_i(j)\in\boldsymbol{{F}}.
