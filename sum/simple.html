Random Variable

A random variable is a variable, a symbol representing a number that can be used in an equation or inequality, and the probability the random variable variable takes on a given value.

Introduce \(X\) for name of random variable.

Discrete: \(P(X = x_j) = p_j\) where \(p_j \ge 0\), \(\sum_j p_j = 1\).

Expected value: \(E[X] = \sum_j x_j p_j\)

Variance: \(\operatorname{Var}(X) = E[(X - E[X])^2] = E[X^2] - E[X]^2\)

Covariance: \(\operatorname{Cov}(X, Y) = E[(X - E[X])(Y - E[Y]) = E[XY] - E[X]E[Y]\)

Probability is a measure: \(P(X\in E\cup F) = P(X\in E) + P(X\in F) - P(X\in E\cap F)\) and \(P(X\in\emptyset) = 0\)

Time Series

A time series is a sequence of random variables indexed by time: \(X_0, X_1, X_2, \ldots\).

Stationary: does not matter when you start observing it.

Uncorrelated: past observations do not influence future observations.

Example: coin flips

White Noise

The most basic time series is white noise

Stationary, uncorrelated, and mean 0

Wold decomposition: every stationary, uncorrelated time series is a moving average of white noise plus a deterministic time series.

Partial Information

Partial information is a partition.

Initially we know only \(\omega\in[0,1)\)

\(\mathcal{A}_0 = \{[0, 1]\}\)

How do we represent "knowing the first digit of \(\omega\)?"

\(\mathcal{A}_1 = \{[0, 1/2), [1/2, 1]\}\)

How do we represent "knowing the first two digits of \(\omega\)?"

\(\mathcal{A}_2 = \{[0, 1/4),\ldots, [3/4, 1]\}\)

Function

If \(A\) and \(B\) are sets then the set exponential \(B^A = \{f\colon A\to B\}\) is the set of all functions from \(A\) to \(B\).

Measure

A measure takes a subset of a set to a number. Measures don't count things twice: the measure of the union of two sets is the sum of their measures minus the measure of their intersection. The measure of the empty set is zero.

The simplest example is counting measure: the number of elements of a set.
